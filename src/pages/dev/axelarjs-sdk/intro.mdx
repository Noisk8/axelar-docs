# AxelarJS SDK

El AxelarJS SDK es una biblioteca npm que incluye una colección de API y herramientas de consulta.

El paquete se puede instalar en tu dApp como una dependencia del proyecto con:
```bash
npm i @axelar-network/axelarjs-sdk
```

Módulos actuales:

- `AxelarAssetTransfer`
    - Se utiliza para la transferencia de tokens entre cadenas mediante la generación de direcciones de depósito.
    - [Transferencia de tokens mediante dirección de depósito](token-transfer-dep-addr).

- `AxelarGMPRecoveryAPI`
    - Biblioteca API para rastrear y recuperar (si es necesario) transacciones GMP (tanto `callContract` como `callContractWithToken`).
    - Las transacciones se indexan por el hash de la transacción iniciada en la cadena de origen al invocar `callContract` o `callContractWithToken`.
    - [Estado de la transacción GMP y recuperación](tx-status-query-recovery).

- `AxelarQueryAPI`
    - Colección de consultas predefinidas útiles en la red, por ejemplo, tarifas de transacción para transferencias de tokens, precios de gas entre cadenas para transacciones GMP, conversiones de denom, etc.
    - [API de consulta de Axelar](axelar-query-api).</document>
